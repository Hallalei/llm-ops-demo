# 技术栈

## 前端框架

- Next.js 16 (App Router)
- React 19
- TypeScript 5.9

## UI/样式

- Tailwind CSS 4
- shadcn/ui (基于 Radix UI)
- Lucide React 图标
- Motion (动画库)
- Recharts (图表)

## 状态/表单

- React Hook Form + Zod 验证
- TanStack Table (数据表格)
- nuqs (URL 状态管理)

## 后端/数据库

- Drizzle ORM + PostgreSQL
- NextAuth.js (认证)

## AI/LLM

- V-API 镜像站 (glm-4-flash)

## 工具链

- pnpm 包管理
- Biome (Lint/Format)
- Docker Compose (本地开发)
